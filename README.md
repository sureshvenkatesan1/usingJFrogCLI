# usingJFrogCLI
Different use-cases for using JFrog CLI and curl to invoke JFrog REST APIs

You need to set these global variables ( I use the /setenv.sh to set these variables):
```
export MYSERVERID=
export MYSERVERHOST_IP=
export PROTOCOL=http
export MYUSER=
export MYPASSWORD=
export MYTOKEN=

```
